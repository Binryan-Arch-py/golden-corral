# Golden Corral

ayuda para gestionar pedidos del negocio de mi padre llamado Golden Corral, dessarrollado íntegramete en **Python**

## Descripcion
Este programa permite gestionar la seleccion de productos y calcular automaticamente el cobro que se le debe hacer al cliente y cuenta con una funcion para mostrar el total de ventas del dia

Con este programa aprendi a usar:
* **Diccionarios** para tener un codigo mas organizado
* **Bases de datos** usé SQLite como base de datos

## Tecnologias
* **Lenguaje:** Python
* **Sistema:** Arch Linux
* **Editor:** VSCode / Neovim

## como ejecutarlo
Para correr este programa, asegurate de tener instalado python y ejecuta:

python golden.py
